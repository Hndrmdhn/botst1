### Alat dan Bahan
Siapin alat dan bahannya.
```bash
> niat
> 2 handphone (1 buat jalanin sc, 1 buat scan qr)
> internet yang memadai, kalo lu gapunya kuota pake kuota kemendikbud ae
> aplikasi whatsapp
> aplikasi termux
> kopi
>rokok
>kalo ada semangat dari doi
```

### Cara Installnya
Sebelum lu jalanin sc nya install dulu lah.
```bash
> kalo lu belum punya apk termux, download di playstore
> masuk ke apk termux lalu ketik dibawah ini!
> $ git clone https://github.com/Hndrmdhn/botst1
> $ cd botst1
> $ bash install.sh
> $ node index.js
> Tinggal scan qr dah
```
